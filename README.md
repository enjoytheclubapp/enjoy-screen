# Enjoy Screen — descargas

Aplicación de escritorio para Windows de **Enjoy the Club**: gestiona la pantalla
del local (peticiones de canciones, fotos, mensajes) y la salida hacia el
proyector / VJ, incluyendo salida **NDI** para Resolume.

Este repositorio contiene **solo el instalador** publicado. El código fuente es
privado.

## Descargar

➡️ **[Descargar para Windows (última versión)](https://github.com/enjoytheclubapp/enjoy-screen/releases/latest/download/Enjoy-Screen-Setup.exe)**

O elige una versión concreta en la pestaña **[Releases](https://github.com/enjoytheclubapp/enjoy-screen/releases)**.

## Requisitos

- Windows 10 / 11 (64 bits).

## Instalación

1. Descarga `Enjoy-Screen-Setup.exe`.
2. Ábrelo y sigue el asistente.

> Mientras la firma de código no esté activa, Windows puede mostrar un aviso de
> SmartScreen ("Windows protegió tu PC"). Pulsa **Más información → Ejecutar de
> todas formas**. Es seguro: el instalador lo publica automáticamente nuestro
> sistema de compilación.
